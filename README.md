# Trabalho_Python
Este trabalho foi feito com base a funcionalidade de um sestema de lan house utilizadado python com django.
Foi feito de uma forma que o usuario e o adiminstrador tenha visão de telas diferentes. APENAS o admin poderá criar, editar e excluir equipamentos e reservas feitas.
O usuario apenas terá permissão para visualizar a tela do Cliente e reservar equipamentos Disponiveis.
Não terá um cadastro de usuarios expecifico ... Ele apenas informará o nome do equipamento que dejezar utilizar, seu nome e seu CPF para identificação (Apenas o admin poderá ver estas informaçoes)

Linguagens e suportes tecnicos:
Foi principamente utilizado a linguagem de Python, com o framework Django e o banco de dados SQlite.
Para a criação dos templates foi utilizado HTML.
Para um visual mais dinamico foi utilizado BootStrap e CSS.
Para tentar ter um resultado mais rapido utilizamos a estenção "Live Share"
A extensão Visual Studio Live Share é uma ferramenta de colaboração em tempo real que permite que desenvolvedores editem e depurem código simultaneamente, independentemente da linguagem de programação ou do sistema operacional
